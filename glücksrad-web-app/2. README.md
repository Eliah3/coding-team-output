# TicTacToe Desktop-Spiel

Ein interaktives Tic-Tac-Toe-Spiel für den Desktop mit KI-Gegner.

## Übersicht

Dieses Projekt ist ein klassisches Tic-Tac-Toe-Spiel mit einer grafischen Benutzeroberfläche und einem computergesteuerten Gegner (KI).

## Funktionen

- 🎮 Spiel gegen den Computer (KI)
- 🖥️ Intuitive grafische Benutzeroberfläche
- 🏆 Gewinn- und Unentschieden-Erkennung
- 🔄 Neustart-Möglichkeit (in Entwicklung)
- 📊 Spielstatistiken (geplant)

## Installation

### Voraussetzungen

- Python 3.8 oder höher
- Tkinter (in Python enthalten)
