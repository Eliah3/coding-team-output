Willkommen bei Tic-Tac-Toe

Spielen Sie gegen den Computer oder einen Freund!

Anleitung

So spielen Sie Tic-Tac-Toe:

  • Es spielen zwei Spieler: X und O
  • Der erste Spieler setzt sein Symbol in ein Feld
  • Der Gegner setzt sein Symbol in ein leeres Feld
  • Wer zuerst drei Symbole in einer Reihe hat, gewinnt

Über das Spiel
