# TicTacToe
Ein einfaches, barrierefreies TicTacToe-Spiel, gebaut mit HTML, CSS und Vanilla JavaScript.

## Starten des Spiels
1. Laden Sie die Dateien herunter oder klonen Sie das Repository.
2. Öffnen Sie die Datei `index.html` in einem beliebigen modernen Webbrowser (Chrome, Firefox, Safari, Edge).

## Funktionen
- **Lokaler Multiplayer:** Spieler X und O wechseln sich am selben Gerät ab.
- **Sieg-Erkennung:** Automatische Erkennung von Reihen, Spalten und Diagonalen.
- **Barrierefreiheit:** Vollständig bedienbar per Tastatur und Screen Reader (ARIA-Labels).
- **Responsiv:** Funktioniert auf Desktop und Mobilgeräten.

## Technologien
- HTML5
- CSS3 (Grid, Flexbox)
- JavaScript (ES6+)
